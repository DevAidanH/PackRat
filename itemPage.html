Description

Polaroid 800 Land Camera, produced between 1957 and 1962 and used type-40 instant roll film that is no longer available. However, if you're handy with tools it can be converted to use 100 pack film, which is still widely available new. This is difficult and Not for the faint of heart. The lens fitted to the Polaroid 800 (and 150) is in fact a very high quality 3-element all-glass coated 130mm F8.8 lens. It will cover 3"x4" with no problem.
